Input To Answer

Messy creative inputs become one test decision.

Forge is designed for the moment before a brand spends more: when the team has ads, copy, screenshots, or performance data, but no clean answer on what should change next. Fewer vague tests. Less budget wasted on trial and error.

InputSource pack

Ad URL, pasted copy, screenshots, CSV, product page, and launch context.

ProcessVariable isolation

Pick the cleanest variable to test and lock the rest of the conditions.

OutputCandidate Brief

Change this. Keep this fixed. Measure this. Prepare the record for the result.

Not A Dashboard

Dashboards show what happened. Forge tells you what to test next.

Ad-level reporting is useful after spend. Forge works before the next spend by converting the result into a controlled experiment decision, so each dollar tests a clearer variable.

Brand Questions

The question is usually not "which report?" It is "what do we change now?"

Forge is written for the questions teams ask before spending more budget, launching another batch, or repeating the same creative debate.

We have ads running, but what should we change next?

Forge turns current creative inputs into one controlled next test.

Why did one ad win if everything changed at once?

Forge separates variables so the next test isolates the likely reason.

Are we wasting budget by testing too many things at once?

Forge keeps the control fixed so spend tests one clearer variable.

Should we test a new hook, format, creator, offer, CTA, or angle?

Forge picks the cleanest variable from the source pack.

Can a small brand start before it has a lot of data?

Yes. Early tests matter more when the decision record starts clean.

How do we stop creative debates from becoming opinion battles?

Forge turns the debate into a test decision, not another vague option list.

Can Forge use Meta, TikTok, Shopify, or Google Ads data?

Yes. Start from URLs, screenshots, copy, CSV exports, product pages, and launch context.

Do we need to connect ad accounts before getting a test decision?

No. A source pack can start before more budget goes live.
